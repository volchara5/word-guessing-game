# Word Guessing Game

A simple console Hangman game written in Python.

The player must guess the hidden word letter by letter.  
Each wrong attempt draws a new part of the hangman.

---

## Features

- random word selection
- hidden word display
- letter checking
- remaining attempts counter
- ASCII hangman drawing
- repeated letter protection

---

## Technologies

- Python 3
- `random` module

---

## Run the project

```bash
python hangman.py
```

---

## Example

```text
Word: - - - - -

Enter a letter: o

Word: - o - - -
```
